# TaglineAIWeb

A simple web application that generates catchy business taglines using **Ollama AI**. Users can enter a business description and select how many taglines to generate (1–5). The app displays clean, ready-to-use taglines.

---

## Demo

Check out the demo video: [TaglineAIWeb Demo](https://drive.google.com/file/d/102IfCRjOR1V8br2cranxlbXTvfN6Prtx/view?usp=sharing)

---

## Features

- Web-based interface (no login required)  
- Generates 1–5 taglines per request  
- Uses Ollama AI `phi` model  
- Clean, modern UI, easy to use  
- Copy taglines to clipboard with one click  

---

## Requirements

- [.NET 8 SDK]
- Visual Studio 2022 or later  
- [Ollama AI](https://ollama.com/) installed locally  
- Minimum 2.5 GB free RAM for the `phi` model
